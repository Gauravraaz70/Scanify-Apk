# 🚀 Scanify — AI Health Product Scanner

> **Scan. Understand. Stay Safe.**

Scanify is an AI-powered mobile-first application that helps users instantly analyze **food products and medicines** using just their camera. It transforms complex nutritional and pharmaceutical data into simple, actionable insights.

---

## 📌 Overview

Scanify solves a real-world problem:
Most people **don’t understand what they consume** — whether it's food ingredients or medicine safety.

With a simple scan, Scanify provides:

* 🥗 Nutritional analysis
* ⚠️ Health risk alerts
* 💊 Medicine safety insights
* 📊 Easy-to-understand health scores

---

## 🎯 Problem We Solve

* Labels are hard to read & understand
* Nutritional data is confusing
* Medicine safety requires expert knowledge
* No unified tool for both food & medicines

👉 **Scanify makes this instant, simple, and accessible.**

---

## ✨ Key Features

### 🥗 Food Analysis

* Barcode-based scanning
* Nutrition breakdown (calories, macros, etc.)
* Allergen detection 🚨
* Ingredient analysis
* Health score (0–100)
* Nutri-score & processing level insights

### 💊 Medicine Analysis

* Label scanning using AI (OCR)
* Usage & dosage information
* Side effects & warnings
* Drug interaction insights
* Safety rating (Safe / Caution)

### 📱 User Experience

* Clean, modern UI
* Dark/light mode 🌙
* Scan history tracking
* Real-time search
* Mobile-first design

---

## ⚙️ How It Works

1. 📷 User scans a product (food or medicine)
2. 🔍 App detects barcode or reads text using AI
3. 🌐 Data is fetched from trusted public sources
4. 🧠 Scanify analyzes and processes the data
5. 📊 Results are displayed in a simple format

---

## 🧠 Core Idea

Scanify combines:

* **Computer Vision (Scanning)**
* **AI-based Text Recognition (OCR)**
* **Health Data Analysis**

👉 To turn raw product data into **meaningful health insights**

---

## 🏗️ Architecture (Simplified)

```
User → Scan → AI Processing → Public Health APIs → Analysis → Result
```

* Fully client-side application
* No heavy backend required
* Fast, scalable, and cost-efficient
